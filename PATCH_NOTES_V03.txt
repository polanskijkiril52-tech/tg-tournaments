Версия 0.3 patch

Что добавлено:
- double elimination (MVP beta): работает для турниров ровно на 4 команды, без grand-final reset
- профиль игрока: display name, роль, bio
- roster management: вступление по invite code, просмотр состава, передача капитана, кик игрока
- турнирные правила / регламент: поле rules_text в турнире + отображение на странице турнира
- history матчей: история завершенных матчей для своей команды
- админ-панель: overview, создание турниров с выбором сетки, быстрый список последних турниров
