Новый патч включает:
- новую горизонтальную сетку турнира в стиле bracket
- кнопку удаления турнира для админа в списке турниров
- кнопку удаления турнира для админа внутри страницы турнира
- использование существующего backend endpoint DELETE /tournaments/{id}

Что заменить:
- webapp/src/App.jsx
- backend/app/routes/tournaments.py

После замены:
backend:
cd backend
uvicorn app.main:app --reload

frontend:
cd webapp
npm run dev

Проверка:
1. зайти под админом
2. открыть список турниров
3. убедиться, что у турниров появилась кнопка Удалить
4. открыть турнир и проверить кнопку Удалить внутри
5. стартовать турнир и открыть страницу турнира
6. проверить, что сетка стала горизонтальной и похожей на bracket-схему
