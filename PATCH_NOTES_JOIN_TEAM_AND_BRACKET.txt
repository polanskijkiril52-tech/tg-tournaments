Что добавлено в патч

1. Игрок может вступить в существующую команду по invite code.
2. У команды теперь есть участники (captain / player).
3. В /teams/me возвращается состав команды, код приглашения и ваша роль.
4. Любой участник команды видит свои матчи; капитан по-прежнему регистрирует команду в турнир.
5. Сетка турнира сохранена и отображается на фронте.
6. Для существующих команд на старте создаётся капитан в team_members и генерируется invite_code.

Новые backend endpoints
- POST /teams/join-by-code
- POST /teams/{team_id}/leave
- GET /teams/{team_id}/members
- GET /teams

Важно
- Капитан не может выйти из команды через leave.
- Для существующих БД добавлена безопасная миграция на старте приложения.
